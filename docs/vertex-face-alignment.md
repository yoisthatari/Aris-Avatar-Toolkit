# Vertex/Face Alignment

Allows selected objects to be aligned to a vertex or the center of a face selected on the active object.

## Usage

1. Select all objects you want to align, leaving the target object as the active object.
2. Enter Edit Mode on the active object.
3. Select the desired vertex or face.
4. Return to Object Mode and open the "Align Tools" panel in the Sidebar.
5. Click "Align by Vertex/Face" and choose the mode (vertex or face).
6. The objects will be aligned to the chosen point.

## Notes

- Works only with Mesh objects.
- Displays error messages if the selection is not valid.
